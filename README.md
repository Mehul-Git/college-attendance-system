# College Attendance & Scheduling System

A role-based college attendance system with Admin, Teacher, and Student dashboards.
Attendance can only be marked from within the campus and during an active attendance window.

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Roles
- Admin: Manages departments, classes, students, teachers, and schedules
- Teacher: Takes attendance and monitors 75% eligibility
- Student: Marks attendance and tracks eligibility
